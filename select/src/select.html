Making Selections

Typically making a selection is easy. Simply tap on your choice and tada it is done.

There are a number of different Jet components that allow for making selections. These include buttonsets (radio and toggle buttons), and select lists (comboboxes).

This page presents simple examples of each of these types of components.

Using VoiceOver with buttons is very easy. Simply double tap the desired selection(s) with a single finger to select it.
